Project Overview

The Solution

This project implements a real-time fatigue monitoring system designed to detect eye-blink patterns using computer vision. The solution continuously monitors user fatigue levels and provides timely alerts to encourage breaks, improving productivity, safety, and user well-being during prolonged screen usage.

The Problem

Extended screen usage often leads to eye strain and fatigue,reducing productivity and increasing health risks. Users usually fail to recognize early signs of fatigue. The absence of a real-time, automated monitoring mechanism creates a gap in preventive intervention.

System Requirements

Functional

  • Capture live webcam input
  • Detect facial landmarks in real time
  • Calculate blink ratio to identify fatigue
  • Classify fatigue levels (Active / Mild / Critical)
  • Display on-screen alerts
  • Trigger audio warning when fatigue increases

Non-Functional

  • Real-time processing with minimal delay
  • High accuracy and consistency
  • Non-intrusive alerts
  • Stable execution during long sessions

Implementation Approach

The system uses MediaPipe Face Mesh to detect facial landmarks from live video input. Eye landmarks are extracted and used to calculate a blink ratio using Euclidean distance. Based on predefined thresholds, fatigue levels are classified and visual/audio alerts are triggered. The system was tested iteratively to ensure smooth real-time performance and usability.
